### Extract Specific Data
```bash
# Get all button selectors
jq '.[] | select(.type == "button") | .selector' ui-elements.json

# Get all form fields
jq '.[] | select(.type | startswith("input_")) | {selector, text}' ui-elements.json

# Get grid data headers
jq '.[0].headers' grid-data.json
```
